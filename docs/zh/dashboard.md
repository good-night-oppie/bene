# BENE 仪表盘 (Dashboard)

只需要一个浏览器标签页，就能让你把整支特种兵大军尽收眼底：谁还在浴血奋战，谁已经凯旋，谁不幸阵亡了 —— 并且能一眼扒出每条进度条背后的所有风吹草动。

> **从一条刺眼的红色报错条，直接追溯到惹祸的那次兵器调用 (tool call) —— 这一切，连你的电脑都不需要出。**

```bash
bene ui           # 唤醒 Web 仪表盘
bene demo         # 塞点假数据并唤醒仪表盘 (不用做任何前置准备)
```

屏幕上你看到的所有东西，全是从本地那个破 SQLite 文件里扒出来的；除非你手贱敲了 `--host 0.0.0.0`，否则这玩意儿死都不会对外网开门。

## 启动花样

```bash
bene ui --port 9000         # 换个门牌号 (默认: 8765)
bene ui --host 0.0.0.0     # 敞开大门，允许外网访问 (用于远程监控)
bene ui --no-browser        # 别特么自动跳出浏览器弹窗
bene ui --db ./project.db  # 强制绑死某个特定的数据库
```

## 怎么看懂时间线

整个画面的绝对C位是一张 **甘特图 (Gantt timeline)**，它把所有的兵按照 *波次 (waves)* 圈了起来：只要是差不多同时起步的特种兵，就会被打包成一波 —— 翻译成人话就是，一次 `bene parallel` 命令，或者一次从 MCP 轰过来的批量任务，算作一波。

### 进攻波次 (Waves)

每一波都有一个领头的表头卡片，上面刻着：

- 终极目标 —— 用大白话写的这波冲锋到底要干嘛
- 发起冲锋的时间，以及鏖战了多久
- 一根被切成了好几段的进度条，按比例涂上了完工的、还在跑的、阵亡的和被强行超度的颜色
- 数人头的小药丸 (pills)

### 进度条 (Bars)

在表头下面，每一只特种兵都化作了一条横杠：

- 长度代表它活了多久，并且按这一波的整体时长做了等比缩放
- 颜色代表死活 —— 绿色是凯旋，紫色是还在冲锋，红色是阵亡，灰色是被叫停或是强行击毙的
- 那些还在跑的条子，会泛着一阵阵的波光
- 一根垂直的 "现在" 游标线，死死咬住活跃波次的当下时刻
- 底下的刻度尺给你一点时间概念 (0s → Xm)

把鼠标悬停在任何一根条子上，就能看到它的老底：代号、职能、领到的任务、以及死活状态。

### 展开与筛选

点一下表头，就能把这波兵折叠或者展开。只要还有兵在跑，或者这一波人数不多 (≤12 个) 的时候，它默认就是敞开的。

顶部的工具栏就是你的狙击镜：

- **搜索框 (search box)** 可以直接拿名字、任务描述或者职能去搜人
- **状态药丸 (status pills)** 让你能把某种状态的人单独拎出来 —— 还在跑的、完工的、阵亡的、被击毙的

## 死死盯住实时战报

画面最底下的那一溜，是一台永远在滚动的事件播报机，只要有风吹草动就会弹出来：谁报错了、谁干完了、谁抽出了兵器、谁往硬盘里塞了文件。表头上的小药丸能帮你把这堆噪音过滤得只剩一种；要是你在这里点了一个特种兵的 ID，右边的审讯室 (inspector) 就会瞬间锁定这个倒霉蛋。

## 提审单只特种兵 (Inspect one agent)

在甘特图里点一根条子 (或者在战报机里点个 ID)，右边就会极其丝滑地滑出一个审讯面板。里面分了 5 个刑房：

**全景扫描 (Overview)** — 这只兵的底裤：死活徽章、什么时候出生的、最后一次喘气是什么时候；记分牌上挂着它写了多少文件、抽了多少次兵器、烧了多少代币、留下了多少条案底；它领到的完整任务书；以及一堆没人看的配置和元数据 JSON。

**流水账 (Events)** — 这只兵这辈子留下的所有脚印，靠着 SSE (Server-Sent Events) 实时怼到你脸上：生死轮回、状态突变、读写了什么文件、拔刀和收刀的瞬间。不用你拿 F5 狂刷，新案底自己就会弹出来。

**兵器谱 (Tool Calls)** — 完整的兵器调用树。每一次挥刀都明明白白写着：它塞了什么进去、拿到了什么结果、磨蹭了多久、以及烧了多少钱的代币；套娃调用会老老实实地缩进显示。点开任何一行，就能看到极其变态的入参和出参细节。

**翻箱倒柜 (Files)** — 这只兵专属的 VFS (虚拟文件系统) 结界的浏览器。你可以踩进目录里，看看文件多大，然后顺着面包屑爬出来。

**时光快照 (Checkpoints)** — 这只兵留下的一串存档点 —— 标签名、存盘时间点、以及几句备忘录。

## 只能用 SSH 或者在 CI 里干瞪眼？

机器上没装浏览器？这套一模一样的数据也能在黑不溜秋的命令行里渲染出来：

```bash
bene dashboard
```

你会得到一张极其硬核的、实时刷新的特种兵存活表，每人烧了多少钱的账单，以及一台能往下滚的事件播报机。

## 在不同的盘子里切来切去

一个仪表盘能管着无数个项目：直接去顶部的下拉菜单里挑另一个 `.db` 文件，画面瞬间就切过去了，连重启都不用。点一下 **+ Project** 就能把任何路径下的数据库生拽进来；在浏览器地址栏里这么搞也是一样的：

`http://localhost:8765/?db=/path/to/bene.db`
