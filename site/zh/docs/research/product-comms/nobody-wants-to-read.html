Nobody Wants to Read Your Sh*t — Steven Pressfield

用途: BENE landing zh rewrite v2 evidence base (post user-rejection 2026-06-13). Lenny bucket: "I want to improve my communication skills" Slug: nobody-wants-to-read · KAOS key: research/product-comms/nobody-wants-to-read Distilled by: workflow wf_3585b934-c3c, parallel subagent with structured schema + web-cited sources.

Thesis

Pressfield 的整本书围绕一个交易关系展开：读者并不亏欠你，他只是太忙、太分心、太被生活揍。每一句没有挣到他注意力的话，都在偷他的时间。Pressfield 在广告业学到的第一课就是 Client's Disease——客户都迷恋自己的产品，以为别人也会爱它；这是任何 landing 文案的原罪。解药是三步：streamline（把信息砍到最简）、make it fun/sexy/scary/informative（让人疯了才会不读）、apply everywhere（小说、TED、grant、给房东的信都一样）。他要求作者把每件作品当一个 transaction：作者必须先想清楚两件事——theme（这玩意到底关于什么）和 concept（一个对常规说法的扭转，一个让其他竞品看起来都没意义的 framing）。结构上必须 Hook → Build → Payoff，必须有 Inciting Incident，必须 stay on-theme。对工程师 landing 而言最致命的提醒是：Nonfiction is fiction——技术文档、布道文也必须按故事来组织，因为读者的注意力机制不在乎你是不是写实。

Reader brain moves

作者引导读者思路的具体动作。每条带概念名 + 怎么做 + 书里的具体例子。

1. 把读者的冷漠先摆到桌面上（Client's Disease 反向操作）

怎么做: Pressfield 不先夸产品，也不先讲方法论；他先承认一个让所有作者难堪的事实——没人想读你写的东西，包括你妈。把这个冷漠当作起点，读者立刻松一口气：终于有人不假装我有义务读完了。这种坦诚反而把读者的注意力请回来。

书里的具体例子: 全书的开篇就是 'Nobody wants to read your shit. They're just busy.' 他不解释为什么这本书值得读，他先把读者的真实状态——忙、不想读——说出来。这种 'I see you' 的姿态是后面所有论证的入场券。

2. 把抽象原则压成一句可重复的口号（concept 先行）

怎么做: Pressfield 反复示范：先找到 concept——'A diamond is forever'、'7Up the Uncola'、'Die Hard on a Plane'——再去展开。concept 不是 slogan，它是一个对常规说法的扭转，让 reader 一秒就 get 到你和别人的差别在哪。他要求作者在动笔前就回答 'What is the concept?'

书里的具体例子: 他用 7Up 'the Uncola' 当案例：直接把 7Up 跟 Coke/Pepsi 摆到平起平坐，而不是说 '我们也是好喝的汽水'。一个扭转的 framing 让所有竞品看起来都 moot and impotent。

3. Nonfiction is fiction——非虚构按故事写

怎么做: Pressfield 反复强调：TED talk、博士论文、grant proposal、给房东的信——全部按 fiction 的结构写。Hook 抓住人，Build 升级紧张感，Payoff 兑现承诺。他不允许作者拿 '我写的是事实' 当不讲故事的借口。

书里的具体例子: 他在书里直接把自己的 The War of Art 拆给读者看：Hook 是 'Here's the problem'（resistance 在你心里），Build 是 'Here's the solution'，Payoff 是改变的承诺。同一个三段式他声称适用于 sales pitch、Master's thesis、军事行动。

4. Inciting Incident——climax 必须埋在开场里

怎么做: Pressfield 要求每个故事的 climax 必须 'embedded within' the Inciting Incident。如果你开场那个让一切发生的事件里，看不到最后的结局，那你根本没有 inciting incident。这逼迫作者从读者第一秒就要看到 stakes。

书里的具体例子: 他对自己作品的复盘里，反复用这个 lens 反向检查：climax 是不是已经在第一场戏里就有了影子？如果没有，就回去重写 inciting incident。

5. 每句自审——读者现在在哪

怎么做: Pressfield 给作者一个可执行的自审清单：每一句都要问自己——'Is this interesting? Is it fun or challenging or inventive? Am I giving the reader enough? Is she bored? Is she following where I want to lead her?' 这把 'empathy' 从一个抽象的德性，变成可以在每一句上调用的检查动作。

书里的具体例子: 这套自审在书里反复出现，Pressfield 用它去解释为什么他每本书要改 10-15 稿——每改一稿就是再走一遍这个清单。

Copyable patterns — 可被 agent 抄走的 5 个句式

每个 pattern 都附带把当前 BENE landing 一条具体 offender 句子改写过的样例。

Pattern 1: Concept-first reframe（先扭转 framing，不解释组件）

句式模板: [对手都在做 X] → [我们的扭转：把 X 变成 Y] → [一句让所有 X 都看起来 moot 的口号]

应用到 BENE landing:

把 'builders.title: 把 Context Engineering 落到一份 SQLite 文件里。' 改写为：'别人的 agent 跑炸了你只能看 log；BENE 跑炸了你可以倒带。Context Engineering 在这里不是哲学，是一个能 cp 走的现场。'

Pattern 2: Hook = 读者此刻的痛点，不是产品的属性

句式模板: [读者凌晨三点正在经历的具体场景] → [他下意识做的那一招] → [那一招失败的瞬间] → [此时 BENE 在场]

应用到 BENE landing:

把 'hero.tagline: 每个 agent 一份隔离 VFS，每一步进 checkpoint，每条声明都能本地重跑。' 改写为：'凌晨三点，Claude Code 把你 prod 的 replicas 从 3 改成了 0。你 git stash，发现它写的是 helm values，stash 救不回来。BENE 在这一刻有用——它从那个 agent 第一次写盘起就在记账。'

Pattern 3: Nonfiction-as-fiction 三段式（Hook → Build → Payoff）

句式模板: [Hook：一个具体事故] · [Build：你试了 N 个标准动作都没救回来] · [Payoff：BENE 用一条命令兑现承诺]

应用到 BENE landing:

把 'rewind.title: 智能体把生产搞挂了。一条命令滚回去。' + 'rewind.body' 合改为：'agent 把 replicas: 3 改成了 0。git stash 没用——它写的是 helm values，不在你 tree 里。git revert 也没用——那不是一次 commit。bene restore 把它写过的每一个字节倒回去，并且告诉你它在哪一 turn、读了什么、想了什么才决定这么改。'

Pattern 4: Streamline——砍到读者能在 7 秒内复述

句式模板: [砍掉所有限定语和形容词] → [一句能被工程师在 Slack 转述的人话]

应用到 BENE landing:

把 'watch.body: 一条命令，零密钥、零注册、不上云。bene demo --no-ui 在本地重放内核的完整故事——trace 被压缩、策略进化一轮、probe 过 kill-gate、越权动作被拒、信任分数被算出。它恰好 0.3 秒跑完，但快不是重点……' 砍成：'bene demo --no-ui。0.3 秒。不用密钥，不联网。你看到的每一个数字在你笔记本上重算了一遍。'

Pattern 5: Stakes-on-the-table（把 reader 的时间成本明码标价）

句式模板: [告诉读者读完这一段他要付出什么] → [告诉他换回的是什么] → [让他自己判断这笔交易划不划算]

应用到 BENE landing:

把 'receipts.title: 你的 tech lead 可以一条命令重跑每一个 check。' 改写为：'你的 tech lead 不需要相信我们。他花 40 秒 clone repo，跑 bene probe verify，每一个 ACCEPT/REJECT 都会带着触发它的那个数值打印出来——sha256 在跑之前就锁了，所以谁都没法改分。如果他不信任这些数，他不需要信任我们。'

Anti-patterns — 作者明确反对的 3 个模式

每个 anti-pattern 引一句当前 BENE landing 的 verbatim 文案做对照，并给出改写。

Anti-pattern 1: Client's Disease（迷恋自己的产品定义，假设读者跟你一样兴奋）

作者为什么反对: Pressfield 在书里明确点名：'There's a phenomenon in advertising called Client's Disease. Every client is in love with his own product. The mistake he makes is believing that, because he loves it, everyone else will too. They won't.' X=Y 式的产品定义（'一个 agent = 一份 SQLite 文件'）正是 Client's Disease 的标本：它假设读者跟你一样觉得 'SQLite' 这三个字母就足以让人激动。读者不在乎你的实现选择。

BENE landing 现在的违规句 (verbatim):

hero.trustLine: '一个 agent = 一份 SQLite 文件。cp 备份它，sqlite3 直接读它，不需要任何 API key，不上云。'

用这本书的纪律改写:

把 '是什么' 换成 '你能用它干什么'：'你的 agent 把 prod 搞挂的那一 turn，留下的是一个文件。cp 一份带走给 tech lead，sqlite3 直接打开看它当时读了什么、想了什么、写了什么。不用 API key，不联网。'

Anti-pattern 2: 没 concept，只有 feature 罗列（违反 'What is the concept?' 铁律）

作者为什么反对: Pressfield 在动笔前的第一个问题是 'What is the concept?'——一个对常规说法的扭转，让所有竞品 framing 看起来都失效。罗列功能（'scratchpad、lessons-learned、跨 session 的持久记忆——这三个之前各自塞在 prompt、临时文件、文档里。BENE 让它们落到同一份 SQLite'）是没 concept 的产物：它告诉读者 BENE 装了什么，但没扭转读者看待 'agent memory' 这件事的 framing。

BENE landing 现在的违规句 (verbatim):

builders.body: 'scratchpad、lessons-learned、跨 session 的持久记忆——这三个之前各自塞在 prompt、临时文件、文档里。BENE 让它们落到同一份 SQLite：checkpoint 就是下一个 agent 能读的 scratchpad；engram 就是跨 session 累积的 lessons-learned；skill 在需要它之前就在 agent 之间传播。能搜、能 diff、能 cp。'

用这本书的纪律改写:

找一个扭转的 framing：'别人在教 agent 怎么记忆，BENE 在教 agent 怎么被审计。下一个 agent 接手时不是读你的 prompt，是读上一个 agent 在哪一 turn 改了哪一行、为什么改、跑没跑过 probe。Memory 在这里不是 context，是证据链。'

Anti-pattern 3: Hook 没抓人——开场说自己的属性，不说读者此刻的处境

作者为什么反对: Pressfield 要求 Hook 必须 grab，且 climax 必须 embedded in the Inciting Incident。'AI 编程智能体跑坏的那一 turn，可以 checkpoint、diff、回滚' 把功能放在前景，把读者的处境（凌晨三点 prod 炸了 git stash 没救回来）藏在了 forWho 副标题里。读者第一眼读到的是产品的能力，而不是自己的痛。Hook 失效。

BENE landing 现在的违规句 (verbatim):

hero.title: 'AI 编程智能体跑坏的那一 turn，可以 checkpoint、diff、回滚。'

用这本书的纪律改写:

把 inciting incident 放到第一句：'Claude Code 把 replicas: 3 改成了 0，prod 炸了。你 git stash，发现它写的是 helm values——stash 救不回来。BENE 让这一 turn 变成可以 restore 的现场。'
