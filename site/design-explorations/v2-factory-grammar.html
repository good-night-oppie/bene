01 The Gap

Four production gaps the research moved past.
Most agent stacks haven't picked them up yet.

Gap 1

Eval scoring ≠ promotion gating

A new prompt or strategy passes an offline eval. That score does not, by itself, refuse to ship a regression — somewhere in your pipeline a human or a deploy script makes the final call. BENE turns the gate into code: promote() raises PromotionBlocked unless a sha256-locked probe says ACCEPT, so CI halts on the exception.

demo · gap-1-killgate
$ bene experiments ls --db $DEMO_DB | grep probe
  → story-probe -> ACCEPT
Gap 2

Cross-agent memory with provenance

Conversation history and single-agent memory blocks are common; what's rare is typed, provenance-linked engrams on a 5-tier ladder that another agent can grep months later and trust. BENE writes every turn as that kind of engram by default.

Gap 3

Autonomy as a computed signal

Most autonomy flags live in a config file — easy to set, easy to forget. BENE folds four signals into a [0,1] trust score, and L4 grants require granted_by='human:<name>' or the API raises. The escalation path is observable, not declarative.

Gap 4

A local file as the source of truth

When an incident review at 03:00 needs the exact prompt that shipped, you want a file on your disk — not a dashboard you're waiting on. BENE's bene.db is that file: cp it, rsync it, diff it, open it with sqlite3.

02 Contract — Four Invariants

Four invariants the harness will not bend on

Before any feature, BENE commits to four things — for every agent, every run, every promotion candidate, every L4 grant.

01
Your agents do not share state by accident
Every agent owns a SQLite-backed virtual filesystem you can diff and restore. Cross-agent reads are explicit; cross-agent decisions ride SharedLog (intent → vote → decide). Isolation is structural — no ACL maintenance.
02
The next agent walks in and greps what the last one learned
Every run leaves typed engrams on a 5-tier ladder: trace 0, episodic 1, semantic 2, procedural 3, strategic 4. Promotion never mutates sources. Provenance is mandatory.
03
A bad prompt cannot ship by passing eval alone
Anything that evolves passes a sha256-locked falsifiable probe before promotion. Tamper the spec, the probe refuses. A gate that can't kill the baseline registers as VOID. Verdicts are themselves engrams, re-readable months later.
04
L4 is the only level a human grants by name
Four signals fold into a [0,1] composite. L4 grants require granted_by='human:<name>' or the API raises. No self-promotion. No YAML knob for it.
